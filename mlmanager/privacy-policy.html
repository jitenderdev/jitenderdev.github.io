Privacy Policy

ML Manager and ML Manager Pro do not collect personal information such as your name, email address, phone number or credit card. It can store file names and paths locally to display them in history lists. It can store other configuration that could improve your experiencie when using the application. ML Manager and ML Manager do not transfer this stored locally information outside of device.

ML Manager and ML Manager Pro use following permissions:

• WRITE_EXTERNAL_STORAGE and READ_EXTERNAL_STORAGE: to read and write files on SD card.
• CHECK_LICENSE: to provide a valid license for the Pro version.
• INTERNET: to send reports about app's crashes and other logs using Crashlytics (Fabric).
